About the website

BookMySlot is a full-stack slot and resource booking website with additional features like self-covid assessment and many other personalized features. This website is basically for students of MIT to allow them to book slots beforehand during this pandemic to avoid over-crowding and have an efficient sports resource allocator. This will also allow interested people to donate sports equipment and money. This website will have user friendly UX and clean UI.

Resources used


1. Java for back-end
2.XML for front-end
3.Firebase realtime database
4.Razorpay for payment gateway
5.covid19india.org for API
